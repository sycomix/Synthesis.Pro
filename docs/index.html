Installation & Setup

Quick Start

  1. Import into Unity
    Download from Unity Asset Store and import into your project
  2. Open Synthesis.Pro
    Unity Menu ‚Üí Window ‚Üí Synthesis ‚Üí Synthesis Pro
  3. Automatic Setup
    Public database downloads automatically on first launch. WebSocket server starts automatically when you open the window
  4. Start Using
    Chat, search, and manage your private knowledge base immediately

Requirements

  • Unity 2020.3 or later
  • Windows, macOS, or Linux
  • Claude API key (from Anthropic) - for AI chat features
  • Python 3.11+ (embedded, included with package)
